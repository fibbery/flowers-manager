
### 方法二：手动启动（分两个终端）

#### 终端 1 - 启动后端

```bash
cd /Users/fibbery/flowers-manager/backend
node server.js
```

看到以下信息表示成功：
```
数据库连接成功
服务器运行在 http://localhost:3000
```

#### 终端 2 - 启动前端

```bash
cd /Users/fibbery/flowers-manager/frontend
npm run dev
```

看到以下信息表示成功：
```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

## 💡 常见问题

### Q: 端口 3000 被占用怎么办？

```bash
# 查找占用进程
lsof -i :3000

# 杀死进程
kill -9 <PID>
```

### Q: 端口 5173 被占用怎么办？

修改 `frontend/vite.config.js` 中的端口号：

```javascript
server: {
  port: 5174,  // 改为其他端口
  ...
}
```
